# My Smart Desk client

React + TypeScript + Vite frontend for the Finance/Document/Policy trackers.

## Scripts
- `npm run dev` – start Vite dev server on port 5173 with `/api` proxy.
- `npm run build` – type-check and build to `dist`.
- `npm run lint` – run ESLint.
- `npm run preview` – preview the built app.

See the root README for environment variables and backend setup.
